Antimicrobial Peptide Prediction Using NLP and Deep Learning Techniques

The paper proposed a novel approach for creating model inputs; using pre-trained language models to produce contextualized embeddings, representing the amino acids within each peptide sequence. A convolutional neural network was used as the classifier, and the performance of both auto-regressive and auto-encoding language models was compared. Test set predictive accuracies outperformed previously published antimicrobial peptide classification models.

Predicting Forest Cover Types Using Neural Computing Methods

Multiclass classification of 7 different forest cover types found in the Roosevelt National Forest, split unevenly across 581k datapoints, using Support Vector Machines (SVMs) and Multilayer Perceptrons (MLPs). Achieved a best overall accuracy score of 93.5% with the MLP model, outperforming previous literature results by 30%.
